# Data Directory

This directory holds the OULAD CSV files. They are not committed to version control due to file size.

## Download Instructions

1. Go to https://www.kaggle.com/datasets/anlgrbz/student-demographics-online-education-dataoulad
2. Download the dataset
3. Place the following CSV files in this directory:

   - `studentInfo.csv`
   - `studentRegistration.csv`
   - `studentVle.csv`
   - `studentAssessment.csv`
   - `assessments.csv`
   - `courses.csv`
   - `vle.csv`

## Citation

Kuzilek, J., Hlosta, M., & Zdrahal, Z. (2017). Open University Learning Analytics Dataset. *Scientific Data*, 4, 170171.
